wir verwenden python-evdev

http://python-evdev.readthedocs.io/en/latest/

The latest stable version of python-evdev can be installed from pypi,
provided that you have gcc/clang, pip and the Python and Linux
development headers installed on your system. Installing them is
distribution specific and typically falls in one of the following
categories:

apt-get install python-dev python-pip 

Mit pip installieren:

https://pypi.org/project/evdev/

pip install evdev 

dann rausfinden welches device die USB Platine ist:

cat /proc/bus/input/devices

dieses device dann in meinem event.py verwenden.

Es steht in der ersten zeile welche python version verwendet wird (evt
anpassen), also kann man es mit .7event.py ausführen.
